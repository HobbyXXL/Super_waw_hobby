## 🔧 Как запустить проект

### 1. Активируй виртуальное окружение (путь примерный, надо вам указать свой путь ниндзя)
cd ~/Рабочий\ стол/HobbyX
source .venv/bin/activate

### 2. Убедись, что PostgreSQL запущен
sudo systemctl start postgresql

### 3. Запусти бэкенд (FastAPI)
cd FASTAPI-MVP
uvicorn main:app --reload

✅ Сервер будет доступен на:
- http://localhost:8000        → приветствие
- http://localhost:8000/docs  → Swagger UI (тестирование API)

### 4. Запусти фронтенд (Next.js)
В новом терминале:
cd ~/Рабочий\ стол/HobbyX/hobbyx-frontend
npm run dev

✅ Фронтенд будет доступен на:
- http://localhost:3000

## ❓ Возникли проблемы?

1. Сервер не запускается? → Проверь, что `.venv` активирован.
2. Ошибки импорта? → Убедись, что все папки содержат `__init__.py`.
3. Нет таблиц в БД? → Выполни `alembic upgrade head`.
4. Ошибка `datetime_type`? → В `schemas/hobbies.py` должно быть:  
   `created_at: Optional[datetime] = None`  
   и импорт: `from typing import Optional`
